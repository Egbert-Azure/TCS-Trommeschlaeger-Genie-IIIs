# This machine: Egbert Schröer's Genie IIIs

Most of `history/` reconstructs software from disk images and other
repositories. This page is different — it's Egbert Schröer's own account
of the specific machine all of that software ran on.

Egbert picked the Genie IIIs up for his friend Claus Ruschinski, on the
same trip he picked up an old Genie I for himself. Claus never really
ended up using it. It was Egbert's own primary machine at the time that
became a TRS-80 Model I, heavily kitted out with add-ons; when that system
finally became too much trouble to keep running, Egbert moved over to the
Genie IIIs instead, and it ran actively until roughly 1995.

The hard disk (a Tandon 10 MB drive, via a Xebec SASI controller) was
fitted early in his ownership — the machine wasn't like that from the
factory, both were add-ons (see `01-format-only-winchester-support/README.md`
for the correction that clarifies this). The real-time clock came later
still; he didn't have it in the first round of add-ons.

Both Egbert and Volker Dose already had to use an IBM PC for business by
this point, which is why being able to read MS-DOS floppies mattered to
them directly — not an abstract compatibility feature, but something they
actually relied on. (See `05-drive-p-alien-formats.md` for what did and
didn't survive of that "alien format" support in the recovered BIOS
sources — the personal motivation described here doesn't resolve that
technical mystery, but it explains why it existed.)

After about 1995 the Genie IIIs mostly sat unused, occasionally booted up
to run dBase programs. Egbert sold it in 2013 when he moved to the US.
Today, Fritz Chwolka — one of the other contributors documented throughout
`history/` — still has it, and still enjoys it.
